# Earth Hero 静态页面导出说明

## 文件结构

- `index.html`：页面入口
- `src/App.js`：主要页面结构和交互逻辑
- `src/main.js`：React 挂载入口
- `src/styles.css`：页面样式
- `src/components/`、`src/globe/`：地球动画相关代码
- `api/baike-home-data.json`：百科数字、历史、热搜数据快照
- `images/`：页面切图资源
- `config/layout.json`：布局参考配置
- `baike-brand-vi.pdf`：按钮点击下载的品牌 PDF
- `server.mjs`：本地静态预览服务
- `scripts/update-baike-data.mjs`：刷新百科数据快照脚本

## 本地预览

进入该目录后运行：

```bash
npm start
```

默认打开：

```text
http://localhost:4173/
```

如果 4173 端口被占用，可使用：

```bash
PORT=4174 npm start
```

## 更新百科数据

如需刷新数字、历史、热搜数据：

```bash
npm run update:data
```

刷新后重新打开页面即可看到新的 `api/baike-home-data.json` 数据。

## 二次调整

- 修改文字和交互：编辑 `src/App.js`
- 修改样式和位置：编辑 `src/styles.css`
- 替换图片：替换 `images/` 内同名文件，或同步修改 `src/App.js` / `src/styles.css` 的图片路径
- 替换下载 PDF：替换 `baike-brand-vi.pdf`

页面使用相对路径，复制整个文件夹到其他设备后，用本地服务打开即可展示。
